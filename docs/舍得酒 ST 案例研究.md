# 舍得酒 ST 案例研究

## 📊 案例背景

### 时间线
- **2021 年**：舍得酒业被 ST（特别处理）
- **ST 原因**：母公司天洋集团资金占用问题，非公司自身经营问题
- **2022 年**：问题解决，申请摘帽
- **摘帽后**：股价飞涨

### 核心逻辑
```
ST 原因：母公司问题（非经营性）
  ↓
公司基本面：良好（白酒行业 + 正常经营）
  ↓
ST 期间：股价被错杀，估值压低
  ↓
摘帽催化：估值修复 + 情绪释放
  ↓
结果：股价飞涨
```

---

## 🔍 关键特征

### 1. ST 原因分类
```
✅ 母公司问题（非经营性）
   - 资金占用
   - 担保问题
   - 股权纠纷
   
❌ 自身经营问题
   - 连续亏损
   - 主营业务萎缩
   - 财务造假
```

### 2. 基本面要求
```
✅ 行业前景好（如白酒）
✅ 公司盈利能力强
✅ 现金流健康
✅ 市场份额稳定
```

### 3. 摘帽催化
```
✅ 问题解决（资金归还/担保解除）
✅ 审计报告无保留意见
✅ 符合摘帽条件
✅ 市场情绪好
```

## 💡 投资策略

### 买入时机
```
最佳时机：
1. 问题接近解决（公告）
2. 摘帽申请已提交
3. 市场情绪低迷（股价低位）

次佳时机：
1. 问题解决公告后
2. 摘帽前 1-3 个月

避免时机：
1. 问题未明朗
2. 摘帽预期已充分炒作
```

### 仓位管理
```
单只仓位：5-10%（高风险）
止损位：-20%（ST 股票波动大）
目标位：+100-300%（摘帽行情）
持有周期：6-18 个月
```

### 风险控制
```
❌ 避免：
- 自身经营问题的 ST
- 财务造假的 ST
- 行业衰退的 ST
- 问题未解决的 ST

✅ 优选：
- 母公司问题的 ST
- 行业景气的 ST
- 基本面良好的 ST
- 问题已解决的 ST
```
