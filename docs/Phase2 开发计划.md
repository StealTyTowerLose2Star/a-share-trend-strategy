## 🔍 高胜率模式提炼

### 分析维度

#### 1. 均线排列
```
完美多头：MA5>MA10>MA20>MA60
部分多头：MA5>MA10 且 MA20>MA60
统计：哪种排列胜率更高？
```

#### 2. 突破幅度
```
突破 1-3%: 胜率？
突破 3-5%: 胜率？
突破 5%+: 胜率？
```

#### 3. 成交量确认
```
放量 150-200%: 胜率？
放量 200-300%: 胜率？
放量 300%+: 胜率？
```

#### 4. 趋势阶段
```
鱼头期（0-30%）: 胜率？
鱼身期（30-70%）: 胜率？
鱼尾期（70-100%）: 胜率？
```

### 提炼方法
```python
1. 分组统计（按不同参数分组）
2. 计算每组胜率
3. 找出胜率>60% 的组合
4. 验证统计显著性
5. 形成高胜率模式
```
