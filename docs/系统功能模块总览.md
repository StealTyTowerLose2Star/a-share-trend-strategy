## 🐲 系统一：龙头战法系统（20% 仓位）

**GitHub**: https://github.com/StealTyTowerLose2Star/a-share-dragon-strategy

**定位**: 高风险、高收益、短线博弈

**预期收益**: 年化 50-100%

### 核心功能模块

#### 1. 数据获取模块（core/limit_up_data.py）
```
功能:
✅ 涨停池数据获取（AKShare）
✅ 跌停池数据获取
✅ 涨停强度统计
✅ 板块涨停统计

数据源:
- AKShare: stock_zt_pool_em
- 连板数、涨停时间、封单金额、所属板块
```

#### 2. 龙头评分模块（core/leader_scorer.py）
```
功能:
✅ 5 维综合评分模型
   - 板块地位（35%）
   - 连板高度（25%）
   - 涨停时间（20%）
   - 带动效应（15%）
   - 抗跌性（5%）
✅ 龙头等级判定
   - 总龙头（≥80 分，7+ 板）
   - 主线龙头（60-79 分，5-6 板）
   - 板块龙头（40-59 分，3-4 板）
   - 补涨龙（<40 分，1-2 板）
✅ 板块龙头识别
```

#### 3. 情绪判断模块（core/market_sentiment.py）
```
功能:
✅ 情绪评分计算（0-100）
✅ 情绪阶段判断
   - 高潮期（80-100）
   - 发酵期（60-80）
   - 启动期/分歧期（40-60）
   - 退潮期（0-40）
✅ 操作建议生成
✅ 情绪报告格式化
```

#### 4. 仓位管理模块（core/position_manager.py）
```
功能:
✅ 动态仓位计算
   - 高潮期：60%（总仓位 12%）
   - 发酵期：80%（总仓位 16%）
   - 分歧期：50%（总仓位 10%）
   - 退潮期：20%（总仓位 4%）
✅ 仓位分配
   - 主线龙头：50%
   - 板块龙头：30%
   - 补涨龙：20%
✅ 单只股票仓位上限
   - 总龙头：≤10%
   - 主线龙头：≤8%
   - 板块龙头：≤6%
   - 补涨龙：≤4%
```

#### 5. 报告生成模块（scripts/generate_report_v5.py）
```
功能:
✅ 情绪周期报告

## 📈 系统二：趋势策略系统（80% 仓位）

**GitHub**: https://github.com/StealTyTowerLose2Star/a-share-trend-strategy

**定位**: 中风险、稳健收益、中线持有

**预期收益**: 年化 25-45%

### Phase 1：核心功能模块

#### 1. 趋势识别模块（core/trend_detector.py）
```
融合理论:
- 海龟交易法则：突破 N 日高点
- 欧奈尔 CANSLIM：平台突破 + 成交量
- 均线多头排列

功能:
✅ 均线多头排列检测（MA5>MA10>MA20>MA60）
✅ 突破 20 日高点确认
✅ 成交量确认（>5 日均量 150%）
✅ 20 日线斜率判断
✅ 趋势强度评分（0-100）
   - 强趋势：≥80 分
   - 中等趋势：60-80 分
   - 弱趋势：40-60 分
✅ 全市场趋势扫描
```

#### 2. 趋势阶段模块（core/trend_stage.py）
```
核心理念:
- 鱼头期（0-30%）：观望，风险高
- 鱼身期（30-70%）：重仓，最确定
- 鱼尾期（70-100%）：轻仓或退出

功能:
✅ 平台整理识别
✅ 趋势阶段计算（基于 120 日高低点）
✅ 回踩确认检测
✅ 买卖信号生成
   - 鱼身期 + 回踩：强烈买入（90% 置信度）
   - 鱼身期：买入（75% 置信度）
   - 鱼身末期：谨慎买入（60% 置信度）
   - 鱼头期：观望（40% 置信度）
   - 鱼尾期：退出（20% 置信度）
```

#### 3. 精确价位模块（core/price_levels.py）
```
融合理论:
- 海龟交易：ATR 动态止损
- 欧奈尔：平台突破买点
- A 股游资：整数关口/前高/均线

功能:
✅ 支撑位计算（前期低点/均线/整数关口/跳空缺口）
✅ 阻力位计算（前期高点/整数关口/百分比目标）
✅ 买入区间计算（理想/激进/保守）
✅ 止损位计算（固定%/ATR/技术位）
✅ 目标位计算（3 档，精确到分）
✅ 风险收益比计算
✅ 交易计划生成

输出示例:
【买入区间】
  理想买入：8.59-8.76 元（回踩平台顶部）
  激进买入：8.67 元（突破确认）
  保守买入：8.20 元（回踩 20 日线）

【止损位】
  止损价格：8.08 元（-5.0%）

【目标位】
  目标 1: 9.35 元（+8.7%）RR=2.0
  目标 2: 9.78 元（+13.7%）RR=4.0
  目标 3: 10.20 元（+18.6%）RR=6.0
```

#### 4. 妖股识别模块（core/yan_gu_detector.py）
```
基于 A 股游资经验

功能:
✅ 连续涨停检测（3 板以上）
✅ 高换手率识别（>15%）
✅ 小市值筛选（<50 亿）
✅ 低股价筛选（<20 元）
✅ 冷门板块爆发识别
✅ 妖股综合评分

评分标准:
- 连续涨停：30 分
- 高换手：20 分
- 小市值：20 分
- 低股价：15 分
- 冷门板块：15 分

>=60 分：妖股
```

#### 5. 市场扫描模块（core/market_scanner.py）
```
功能:
✅ 全市场 A 股代码获取（5000+ 只）
✅ ST 股票过滤
