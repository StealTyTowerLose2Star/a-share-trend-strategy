# A 股趋势策略系统

专业的中期趋势跟踪系统，基于全市场扫描和右侧交易原则，捕捉趋势中段收益。

## 🎯 核心理念

### 四大原则
1. **右侧交易** - 不预测趋势，只跟随已形成的趋势
2. **中段收益** - 放弃鱼头鱼尾，只吃最确定的 30-70%
3. **全局判断** - 基于全市场 5000+ 股票数据统计
4. **精确建议** - 买入/卖出/止损价位精确到分

### 融合经典理论
- **海龟交易法则** - 突破 N 日高点 + ATR 止损
- **欧奈尔 CANSLIM** - 价格平台突破 + 成交量确认
- **理查德·丹尼斯趋势跟踪** - 让利润奔跑 + 快速止损
- **A 股游资经验** - 妖股特征 + 龙虎榜 + 情绪周期

## 🚀 核心功能

### 1. 趋势识别（右侧确认）
- 均线多头排列（MA5/10/20/60）
- 价格突破关键阻力
- 成交量确认（>5 日均量 150%）
- 趋势强度评分

### 2. 趋势阶段判断
- 鱼头：突破初期（0-30%）→ 观望
- 鱼身：趋势确认（30-70%）→ 重仓
- 鱼尾：趋势末期（70-100%）→ 轻仓或退出

### 3. 全市场扫描
- 每日扫描 5000+ 只股票
- 自动识别趋势形成
- 妖股特征捕捉
- 冷门板块机会

### 4. 精确交易计划
- 买入区间：精确到分
- 止损位：精确到分
- 目标位：3 档精确价位
- 仓位建议：百分比

## 📈 趋势识别标准

### 中期趋势形成信号

```python
# 必要条件（全部满足）
1. MA5 > MA10 > MA20 > MA60（均线多头排列）
2. 收盘价 > 60 日高点（突破关键阻力）
3. 成交量 > 5 日均量 150%（成交量确认）
4. 20 日线斜率 > 0（趋势向上）

# 充分条件（满足越多越强）
1. RSI(14) 在 50-70 之间（强势区）
2. MACD 金叉且在水上
3. 相对强度 RS > 80（跑赢 80% 股票）
4. 所属板块为热门主线
```

### 趋势强度评分

```python
趋势评分 = (
    均线排列分 * 30 +    # 多头排列完整性
    突破强度分 * 25 +    # 突破幅度
    成交量分 * 20 +      # 成交量确认
    相对强度分 * 15 +    # 跑赢大盘
    板块热度分 * 10      # 板块效应
)

评分 >= 80: 强趋势（重仓）
评分 60-80: 中等趋势（标准仓）
评分 40-60: 弱趋势（轻仓）
评分 < 40: 观望
```

## 🎯 交易规则

### 买入规则

```python
# 标准买入（回踩确认）
if 趋势形成 and 
   回踩 20 日线不破 and 
   成交量萎缩后再次放大:
    买入 50% 仓位

# 激进买入（突破确认）
if 趋势形成 and 
   突破平台 and 
   成交量 > 5 日均量 200%:
    买入 30% 仓位
```

### 止损规则

```python
# 初始止损
止损位 = min(买入价 * 0.95, 20 日线 * 0.97)

# 移动止损（盈利后）
if 盈利 > 10%:
    止损位 = 最高价 * 0.90  # 回撤 10% 止损
elif 盈利 > 20%:
    止损位 = 最高价 * 0.85  # 回撤 15% 止损
```

### 止盈规则

```python
# 目标位止盈
目标 1 = 买入价 * 1.08  # 减仓 30%
目标 2 = 买入价 * 1.15  # 减仓 30%
目标 3 = 买入价 * 1.25  # 清仓

# 趋势破坏止盈
if 收盘价 < 20 日线:
    清仓
```

## 📊 仓位管理

### 单只股票仓位

```python
if 趋势评分 >= 80:
    单只仓位 = 15%  # 强趋势
elif 趋势评分 >= 60:
    单只仓位 = 10%  # 中等趋势
else:
    单只仓位 = 5%   # 弱趋势
```

### 总仓位控制

```python
# 根据市场环境
if 大盘趋势向上 and 涨停数 > 80:
    总仓位 = 80%
elif 大盘震荡:
    总仓位 = 50-60%
else:
    总仓位 = 20-30%
```

## ⚠️ 风险提示

1. **趋势失败风险** - 假突破可能导致亏损
2. **震荡市风险** - 趋势策略在震荡市表现不佳
3. **数据延迟风险** - 免费数据源可能延迟
4. **黑天鹅风险** - 极端行情下策略可能失效
